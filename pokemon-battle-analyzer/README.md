# 宝可梦冠军 · 对战分析工具

面向《宝可梦 朱/紫》冠军赛规则（Lv.50 双打）的浏览器端伤害计算与对战分析工具。

## 功能

- **队伍配置**：选择宝可梦、设置努力值/性格/道具/特性/招式，支持多队伍保存
- **伤害计算**：计算我方招式对对方三种防御配置的伤害范围（%HP）
- **威胁分析**：列出对方可学招式对我方的威胁伤害，按威胁度排序
- **场地支持**：天气、场地、光墙/反射壁、顺风等场地状态修正
- **特性/道具**：主流进攻/防御特性与强化道具的伤害修正

## 特殊伤害计算

| 招式 | 计算方式 |
|------|----------|
| 扑击 | 使用使用者**防御**代替攻击计算伤害 |
| 欺诈 | 使用**目标的攻击**代替使用者攻击计算伤害 |
| 光子喷涌 / 太晶爆发 | 使用攻击与特攻中**较高者**计算伤害 |
| 硬压 | 威力 = 目标剩余HP百分比（最高100） |
| 扫墓 | 威力 = 50 + 50×倒下队友数（上限300） |
| 愤怒之拳 | 威力 = 50 + 50×被攻击次数（上限350） |
| 全开猛撞 / 闪电猛冲 | 超效时伤害额外×1.3 |
| 陀螺球 | 威力 = floor(25 × 对方速度 / 我方速度)，上限150 |
| 电球 | 威力按我方/对方速度比分档（40/60/80/120/150） |
| 打草结 / 踢倒 | 威力按目标体重分档（20/40/60/80/100/120） |
| 重磅冲撞 / 热力碰撞 | 威力按我方/目标体重比分档（40/60/80/100/120） |
| 喷水 / 巨龙威能 / 喷发 | 满HP威力150，随HP降低而减弱 |
| 辅助力量 / 能量球 | 威力 = 20 + 20×正面能力提升级数 |

## 文件结构

```
pokemon-battle-analyzer/
├── index.html          # 主页面
├── style.css           # 样式
├── app.js              # 主逻辑（UI 交互、队伍管理）
├── calc/
│   └── damage.js       # 伤害计算核心
└── data/
    ├── moves.js        # 招式数据库（~600 条）
    ├── showdown_cn.js  # 招式/特性/道具中文名
    └── Moves.txt       # 招式原始数据（制表符分隔）
```

## 使用方式

直接用浏览器打开 `index.html`，无需构建或服务器。

## 技术说明

- 纯原生 JS，无框架依赖，所有数据通过 `<script>` 标签作为全局变量加载
- 伤害公式：Gen V+（Lv.50）`floor(floor(22 × Power × A / D) / 50) + 2`，随机数范围 0.85–1.00
- 命中要害：×1.5，无视攻击方负面能力变化及防御方正面防御/特防变化
